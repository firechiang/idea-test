#### 一、下载IDEA-2019.2.4软件包（百度网盘）
#### 二、下载IDEA-2019.2.4破解工具]（百度网盘）
#### 三、解压IDEA软件包
![image](https://github.com/firechiang/kubernetes-study/blob/master/jenkins/image/idea100.png)
#### 四、解压IDEA-2019.2.4破解工具（建议将破解文件拷贝到IDEA的解压目录）
![image](https://github.com/firechiang/kubernetes-study/blob/master/jenkins/image/idea101.png)
#### 五、找到IDEA-2019.2.4解压目录下的“bin”目录下的“idea64.exe”双击文件打开IDEA
![image](https://github.com/firechiang/kubernetes-study/blob/master/jenkins/image/idea102.png)
#### 六、选择是否导入配置（建议选择不导入）
![image](https://github.com/firechiang/kubernetes-study/blob/master/jenkins/image/idea103.png)
#### 七、后面请一路next直到激活界面，选择试用
![image](https://github.com/firechiang/kubernetes-study/blob/master/jenkins/image/idea104.png)
#### 八、进入欢迎页面，点击“configure”，再点击“Edit Custom VM Options”
![image](https://github.com/firechiang/kubernetes-study/blob/master/jenkins/image/idea105.png)
#### 九、在最后加入一行“-javaagent:激活文件绝对路径”（注意：这里就是指定破解文件所在目录）
![image](https://github.com/firechiang/kubernetes-study/blob/master/jenkins/image/idea106.png)
#### 十、关闭idea，关闭idea，关闭idea，重要的事说三遍，重新打开，进入欢迎页面，点击“configure”，再点击“Manage License”，将破解工具中的激活码文件内容全部复制，粘贴到idea中，点击OK
![image](https://github.com/firechiang/kubernetes-study/blob/master/jenkins/image/idea107.png)
#### 十一、至此IDEA-2019.2.4就破解好了
![image](https://github.com/firechiang/kubernetes-study/blob/master/jenkins/image/idea108.png)
