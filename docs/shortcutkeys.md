#### 一、快捷键简单说明
```bash
Ctrl + D        # 复制并生成一行代码
Ctrl + Y        # 删除一行代码
A|t + Enter     # 导包或自动添加变量名
Ctrl + A|t + T  # 为一段代码加try或if等代码块
A|t + Insert    # 自动生成函数，比如Set或Get等等
```
